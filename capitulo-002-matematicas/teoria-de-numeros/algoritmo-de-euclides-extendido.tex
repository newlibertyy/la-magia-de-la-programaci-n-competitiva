\subsection{algoritmo de Euclides extendido}
el algoritmo de Euclides extendido sirve para hallar 2 n√∫meros t y s que dados a y b\\
$at+bs=gcd(a,b)$
\begin{lstlisting}[style=C,caption=euclidesExtendido.cpp]
int x,y,d;
void euclidesExtendido(int a, int b) {
    //caso base
	if (b == 0) {
        x = 1;
        y = 0;
        d = a;
        return;
    }
	euclidesExtendido(b, a % b);
	int x1 = y;
	int y1 = x - (a / b) * y;
	x = x1;
	y = y1;
}
\end{lstlisting}
